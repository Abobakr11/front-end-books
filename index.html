الدرس الأول: التعليقات في HTML


  


  

    
في هذا الدرس نتعلم التعليقات في HTML.

    
التعليقات تبدأ بـ <!-- وتنتهي بـ -->.

    
كل شيء بينهما لن يتم تنفيذه من المتصفح، ويمكن استخدامها لكتابة ملاحظات أو شروحات.

  







  
الدرس الثاني: مستند HTML أساسي


  

    

  


  

    
هذا الدرس يوضح الهيكل الأساسي لأي صفحة HTML.

    
يتضمن تعريف نوع المستند <!DOCTYPE html>، الرأس <head>، والعنوان <title>.

    
كما يحتوي الجسم <body> لعرض محتوى الصفحة.

  







  
الدرس الثالث: HTML مع ميتا و CSS و JavaScript


  

    

  


  

    
هذا الدرس يوضح كيفية إضافة إعدادات الترميز باستخدام <meta charset="UTF-8">.

    
كما يوضح مكان إضافة أكواد CSS و JavaScript وروابط خارجية.

  







  
الدرس الرابع: العناوين في HTML


  

    

  


  

    
يشرح هذا الدرس كيفية استخدام وسوم العناوين من <h1> إلى <h6>.

    
العناوين الأكبر لها تأثير أكبر في الصفحة وتقل تدريجيًا حسب الرقم.
