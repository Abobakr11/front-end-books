الدرس الأول: مستند HTML أساسي



    
    

<!DOCTYPE html>
- يُعرّف نوع المستند ويخبر المتصفح أننا نستخدم HTML5.

<html lang="ar"> ... </html>
- العنصر الجذري للمستند. سمة lang="ar" تُشير إلى أن اللغة عربية.

<head> ... </head>
- قسم الميتاداتا: يحتوي <meta>، <title>، وصلات إلى ملفات CSS/JS.

<meta charset="utf-8">
- يحدد ترميز الحروف (UTF-8) لعرض النص العربي والرموز بشكل صحيح.
<title>...</title>
- عنوان الصفحة الذي يظهر في تبويب المتصفح.

<body> ... </body>
- يحتوي المحتوى المرئي للمستخدم (نصوص، صور، أزرار...).
    

  


  

    
الدرس الثاني: التعليقات في HTML

    


    

مثال التعليق في HTML:
<!-- هذا تعليق داخل الجسم -->

- التعليقات لا تظهر في واجهة المستخدم لكنها تبقى في مصدر الصفحة.
- تُستخدم لترك ملاحظات للمطورين أو تعطيل جزء من الكود مؤقتًا.
    

  


  

    
الدرس الثالث: HTML مع ميتا و CSS و JavaScript

    


    

<meta charset="UTF-8">
- يحدد ترميز الصفحة (مهم للنصوص متعددة اللغات).

<link rel="stylesheet" href="style.css">
- يربط ملف CSS خارجي لتنسيق الصفحة.

<script src="script.js"></script>
- يضمّن ملف JavaScript خارجي. يتم وضعه عادةً قبل نهاية </body> أو في <head> حسب الحاجة.
    

  


  

    
الدرس الرابع: العناوين في HTML

    


    

<h1> ... </h1>
- عنوان المستوى الأول (الأهم).

<h2> ... </h2> إلى <h6> ... </h6>
- عناوين بمستويات أقل لتنظيم المحتوى.
- تُستخدم لأغراض دلالية (SEO، سهولة القراءة)، وليس فقط للتنسيق.
    

  


  

    
الدرس الخامس: الفقرات في HTML

    


    

<p> ... </p>
- عنصر الفقرة لاحتواء نص.
- المتصفح يضيف مسافة (فاصل) قبل و بعد كل <p>…</p> فقرة تلقائيًا.
- للفواصل السطرية بدون بداية فقرة جديدة استخدم <br>.
